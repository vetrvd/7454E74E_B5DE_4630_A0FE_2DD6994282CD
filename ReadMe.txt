Задание.

1. Авторизация пользователей
2. API для:
  - Создания поста
  - Получение постов с использованием фильтров (по пользователю, по тегам, по строковому поисковому запросу)
Технические требования:
1. Язык - C#
2. Платформа - .Net Core 2.1
3. База данных MongoDB

Реализация.

1. Авторизация - для авторизации используются jwt токены. Для запуска приложения и получения валидного токена нужно 
    a) правильно указать ClientSecret 
        *или в appconfig 
        *или в commandline 
        *или в environment 
        *или в user-secret секции (dotnet user-sercrets --id '6B0846D5-2CC0-4E99-A6B4-1AAEC6634E69' set ClientSecret 'value')
    б) получить валидный токен, для тестовых нужд можно использовать сайт https://jwt.io/ (Algorithm hs256 и Указать ClientSecret) 

2. Реализовано простейшее api 
    a) получить "все" записи GET https://localhost:5001/api/note
    б) получить пост по ключу GET https://localhost:5001/api/note/5ba177c7d000790148919fa3
    в) использование фильтра GET https://localhost:5001/api/note/?User=member&Text=text&tag=tag
    г) добавление новой записи POST https://localhost:5001/api/note
